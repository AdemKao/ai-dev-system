# Commit Command

Create a commit following conventional commit standards.

## What This Does

1. Check git status and staged changes
2. Analyze changes to determine commit type
3. Generate conventional commit message
4. Create the commit

## Usage

```
/commit                    # Commit staged changes
/commit --all              # Stage all and commit
/commit "message"          # Use provided message
```
