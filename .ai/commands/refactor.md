# Refactor Command

Refactor code safely while maintaining behavior.

## What This Does

1. Ensure tests exist for target code
2. Apply refactoring techniques
3. Verify tests still pass
4. Commit changes atomically

## Usage

```
/refactor <file>           # Refactor specific file
/refactor --extract        # Extract methods/components
/refactor --rename         # Rename symbols
```
