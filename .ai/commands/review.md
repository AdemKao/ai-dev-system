# Review Command

Perform a thorough code review following best practices.

## What This Does

1. Load code review workflow and skill
2. Analyze changes for quality, security, and maintainability
3. Provide actionable feedback

## Usage

```
/review                    # Review staged changes
/review <file>             # Review specific file
/review --security         # Focus on security aspects
/review --performance      # Focus on performance
```
