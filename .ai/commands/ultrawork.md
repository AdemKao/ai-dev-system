# Ultrawork Command

Enable maximum orchestration mode with parallel agent execution.

## What This Does

1. Activate orchestrator agent
2. Enable parallel task execution
3. Leverage multiple specialist agents
4. Continue until task is complete

## Usage

```
/ultrawork <task>          # Execute complex task
/ulw <task>                # Shorthand
```

## Behavior

When ultrawork is activated:
- Tasks are broken into subtasks
- Subtasks are delegated to specialists
- Multiple agents work in parallel
- Progress is tracked continuously
- Task completion is enforced
